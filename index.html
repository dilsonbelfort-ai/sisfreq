Configurar Firebase

Complete estes passos para ativar o banco de dados

1
Criar projeto no Firebase
Acesse console.firebase.google.com, clique em "Adicionar projeto" e siga os passos. É totalmente gratuito.
2
Ativar Firestore Database
No painel do projeto → Firestore Database → "Criar banco de dados" → selecione Modo de teste (30 dias gratuito, depois configure as regras).
3
Registrar app web e copiar config
Em Configurações do projeto → "Seus aplicativos" → ícone Web (</>) → copie o objeto firebaseConfig e substitua no trecho abaixo dentro do arquivo index.html:
const firebaseConfig = { apiKey: "SUA_API_KEY_AQUI", authDomain: "SEU-PROJETO.firebaseapp.com", projectId: "SEU-PROJECT-ID", storageBucket: "SEU-PROJETO.appspot.com", messagingSenderId: "000000000000", appId: "1:000000000000:web:xxxxxxxxxxxxxxxx" };
4
Publicar no GitHub Pages
Faça upload do index.html em um repositório GitHub → Settings → Pages → Source: Deploy from branch → Branch: main → Save. Em 1–2 minutos seu sistema estará online.
